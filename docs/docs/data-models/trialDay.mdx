# Dogs Have Owners

This is an individual day of a trial, which is associated with a Trial itself. For example, Friday, Saturday and Sunday each have their own info below

| Property        |   Data Type    | Notes                                                                                                                                                     |
| :-------------- | :------------: | :-------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Date            |     string     |                                                                                                                                                           |
| Name of Day     |     string     | default to day of week                                                                                                                                    |
| Day's AKC id    |     string     | provided by AKC                                                                                                                                           |
| Trail Secretary | trialSecretary |                                                                                                                                                           |
| Judges          |  array of ids  |                                                                                                                                                           |
| Entry Limit     |     string     | The max number of RUNS that can happen that day PER JUDGE. The max is 350                                                                                 |
| Mixed Breeds    |    boolean     |                                                                                                                                                           |
| Rings           |      Ring      | A ring is analagous to a baseball field. Some trials have one ring, with one course going during the day. Others have two rings operating simulataneously |
