# Tourney Time

Light that baby, up!

| Property     |  Data Type   | Notes                                                                                                                                                      |
| :----------- | :----------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Date         |     date     |                                                                                                                                                            |
| Name of Day  |    string    | Default to day of week                                                                                                                                     |
| Day's AKC ID |    string    | Provided by AKC                                                                                                                                            |
| Judges       | array of ids |                                                                                                                                                            |
| Entry Limit  |    number    | The max number of RUNS that can happen that day PER JUDGE. Maximum: 350.                                                                                   |
| Mixed Breeds |   boolean    |                                                                                                                                                            |
| Rings        |     Ring     | A ring is analagous to a baseball field. Some trials have one ring, with one course going during the day. Others have two rings operating simulataneously. |
